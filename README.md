# Инструкция по запуску
1. Устанавливаем виртуальное окружение в папке проекта и запускаем
```
python -m venv env
```

```
Windows:
cd env/Scripts/
activate.bat
```

2. Устанавлиаем зависимости
```
pip install -r req.txt
```

3. Запускаем программу
```
python main.py
```
